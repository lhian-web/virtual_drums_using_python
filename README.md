# Virtual Drums

### Project Overview
The **Virtual Drums** is an interactive drum kit that lets you play drums using a blue colored object in front of your webcam. It uses **OpenCV** to detect the object and **Pygame** to play drum sounds. When your object hits a virtual drum pad, the corresponding sound is played, and the drum image appears on screen, giving a real-time, immersive experience.

### Features
- **Object Detection:** Tracks a blue-colored object in real time using your webcam.
- **Interactive Drum Pads:** Play different drum sounds by moving the object over virtual pads.
- **Visual Feedback:** Drum images overlay on the video feed for a realistic effect.
- **Customizable:** Add more drums, change colors, or use your own drum images and sounds.
- **Easy to Use:** Run the Python script and start drumming—no special hardware needed.

### Requirements
- Python 3.11
- OpenCV (`cv2`)
- NumPy (`numpy`)
- Pygame (`pygame`)
